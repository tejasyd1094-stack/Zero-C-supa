This repository contains a production-ready Next.js (App Router) app integrated with Supabase for:
- Passwordless email auth (magic link)
- Usage tracking (3 free trials)
- Optional OpenAI script generation

## Quick start (non-developer)
1. Upload this repo to GitHub.
2. Create a Supabase project and run `supabase.sql` in SQL editor.
3. Add environment variables in Vercel (see `.env.example`).
4. Add images to `public/`: `logo.png`, `splash-light.png`, `splash-dark.png`.
5. Deploy to Vercel (import repo) and redeploy after env vars set.

See more in the project docs.
